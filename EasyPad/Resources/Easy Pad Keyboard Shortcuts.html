Help (F1)

You may use the keyboard shortcut for Help (F1) anytime there is a “Help” button visible. The appropriate Help HTML page will open in your preferred browser.

View Shortcuts

View ► Theme Editor (F11)

Zoom Shortcuts

Zoom ► Zoom In (CTRL++) — use either the +/= key or the plus key on the number pad

Zoom ► Zoom In (CTRL+–) — use either the _/– key or the minus key on the number pad
